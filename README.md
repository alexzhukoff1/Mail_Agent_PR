# Personal Secure Email Audit Agent

Read-only Telegram-бот для анализа почты из нескольких провайдеров (Gmail, Yandex, Mail.ru и др.).

## Принципы

- **Read-only** — никаких действий с письмами (ответ, пересылка, открытие ссылок)
- **Security first** — изоляция сети, шифрование, аудит
- **Learning** — обучение на обратной связи пользователя

## Структура проекта

- `Full_System_Prompt.txt` — полная спецификация архитектуры
- `.cursor/rules/` — правила для Cursor AI
- `.cursor/skills/` — скиллы для разработки

## Разработка

Проект в стадии спецификации. См. `Full_System_Prompt.txt` для деталей архитектуры.
